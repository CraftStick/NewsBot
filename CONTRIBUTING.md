# Contributing

Спасибо за интерес к проекту.

## Как предложить изменения

1. Форкните репозиторий
2. Создайте ветку: `git checkout -b feature/my-change`
3. `make check` должен проходить
4. Откройте Pull Request с кратким описанием «зачем»

## Стиль

- Минимальный diff — только то, что нужно для задачи
- Комментарии — по делу, на русском или английском
- Секреты и `.env` в PR не добавлять

## Идеи для PR

- Новые RSS-ленты с обоснованием
- Улучшение устойчивости Gemini API
- Документация и переводы

Вопросы — через [Issues](https://github.com/CraftStick/NewsBot/issues).

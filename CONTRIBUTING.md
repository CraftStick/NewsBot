# Contributing

[English](#english) · [Русский](#русский)

## English

Thanks for your interest in this project.

### How to contribute

1. Fork the repository
2. Create a branch: `git checkout -b feature/my-change`
3. Run `make check` — it must pass
4. Open a Pull Request with a short description of **why** the change is needed

### Guidelines

- Keep diffs focused — only what the task requires
- Comments in English or Russian are fine
- Do not include secrets or `.env` files in PRs

### Good PR ideas

- New RSS feeds (with a short rationale)
- Gemini API resilience improvements
- Documentation and translations

Questions: [Issues](https://github.com/CraftStick/NewsBot/issues)

## Русский

Спасибо за интерес к проекту.

### Как предложить изменения

1. Форкните репозиторий
2. Создайте ветку: `git checkout -b feature/my-change`
3. `make check` должен проходить
4. Откройте Pull Request с кратким описанием «зачем»

### Стиль

- Минимальный diff — только то, что нужно для задачи
- Комментарии — по делу, на русском или английском
- Секреты и `.env` в PR не добавлять

### Идеи для PR

- Новые RSS-ленты с обоснованием
- Улучшение устойчивости Gemini API
- Документация и переводы

Вопросы — через [Issues](https://github.com/CraftStick/NewsBot/issues).
